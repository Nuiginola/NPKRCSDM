# NPK RC SDM

โปรแกรมออกแบบโครงสร้างคอนกรีตเสริมเหล็กด้วยวิธีกำลัง (Strength Design Method)
ตามกฎกระทรวง พ.ศ. 2566 / อิง ACI 318 — เวอร์ชัน 1.3.2

## โมดูล
- **พื้น:** 1.1 พื้นวางบนดิน, 1.2 พื้นทางเดียว, 1.3 พื้นสองทาง, 1.4 พื้นยื่น
- **บันได:** 2.1 บันไดช่วงตรง, 2.2 บันไดหักกลับ (U-Shape)
- **คาน:** 3.1 คานช่วงเดียว, 3.2 คานต่อเนื่อง, 3.3 คานยื่น
- **เสา:** 4.1 เสาสี่เหลี่ยม, 4.2 เสากลม
- **ฐานราก:** 5.1 ฐานรากแผ่, 5.2 ฐานรากเสาเข็ม (Pile Cap)

## การรันโปรแกรม (โหมดพัฒนา)
```
pip install -r requirements.txt
python desktop_app.py
```
เปิดเป็นหน้าต่างโปรแกรม (pywebview + WebView2) ไม่ใช่เว็บเบราว์เซอร์

## เทคโนโลยี
Streamlit + pywebview (WebView2) + matplotlib + Pillow
