# NPK RC SDM — เวอร์ชันคลาวด์ (ใช้บนมือถือ/แท็บเล็ต/iOS/Android)

โฟลเดอร์นี้คือชุดโค้ดสำหรับนำขึ้น **Streamlit Community Cloud (ฟรี)** เพื่อเปิดใช้โปรแกรม
ผ่านเบราว์เซอร์บนมือถือ/แท็บเล็ตได้ทุกที่ **แยกต่างหากจากโฟลเดอร์ `code` เดิม ไม่กระทบเวอร์ชันเดสก์ท็อป**

> เวอร์ชันคลาวด์นี้ **ไม่มีหน้า login** (เปิดปุ๊บใช้ได้ทันที) ตามที่ตั้งใจไว้ว่าใช้ส่วนตัวไว้ตรวจงานนอกโต๊ะ
> ส่วนเวอร์ชันเดสก์ท็อป (Windows) ยังถามรหัสเหมือนเดิม

---

## ขั้นตอนนำขึ้นคลาวด์ (ทำครั้งเดียว ~10–15 นาที)

### 1) สมัครบัญชี (ฟรี ทั้งคู่)
- **GitHub** — https://github.com/signup
- **Streamlit Community Cloud** — https://share.streamlit.io  (กด Sign in with GitHub ได้เลย)

### 2) สร้าง Repository บน GitHub แล้วอัปโค้ด
วิธีง่ายสุด (ผ่านหน้าเว็บ ไม่ต้องใช้คำสั่ง git):
1. ที่ GitHub กด **New repository** → ตั้งชื่อ เช่น `npk-rc-sdm` → เลือก **Private** (แนะนำ) → **Create**
2. ในหน้า repo กด **uploading an existing file**
3. ลาก **ไฟล์และโฟลเดอร์ทั้งหมดในโฟลเดอร์นี้** (app.py, requirements.txt, common/, app_pages/, modules/, data/, .streamlit/ ฯลฯ) เข้าไปวาง
4. กด **Commit changes**

### 3) Deploy บน Streamlit Cloud
1. เข้า https://share.streamlit.io → **Create app** → **Deploy a public app from GitHub**
2. เลือก **Repository** = repo ที่เพิ่งสร้าง, **Branch** = main, **Main file path** = `app.py`
3. กด **Deploy** → รอสัก 2–5 นาที (ครั้งแรกติดตั้งแพ็กเกจ)
4. เสร็จแล้วจะได้ลิงก์ เช่น `https://npk-rc-sdm.streamlit.app`

### 4) เปิดบนมือถือ / iPad / iPhone / Android
- เปิดลิงก์ในเบราว์เซอร์ (Safari/Chrome)
- อยากให้เหมือนแอป: กด **แชร์ → เพิ่มลงในหน้าจอโฮม (Add to Home Screen)** จะมีไอคอนเปิดได้เลย

## ข้อควรรู้
- แอปฟรีจะ **"หลับ" เมื่อไม่มีคนใช้สักพัก** เปิดครั้งถัดไปรอ ~30 วินาทีให้ตื่น ถือเป็นเรื่องปกติ
- ทรัพยากรมีจำกัด (เหมาะกับการตรวจงาน/คำนวณทั่วไป ไม่ใช่งานหนักพร้อมกันหลายคน)
- ข้อมูลที่บันทึกระหว่างใช้งานเป็นแบบชั่วคราว (เซิร์ฟเวอร์รีสตาร์ตแล้วหาย) — ถ้าต้องเก็บถาวรให้ดาวน์โหลดรายการคำนวณเป็น PDF ไว้
- ฟอนต์ไทยในกราฟใช้ไฟล์ที่ฝังมาในโฟลเดอร์ (`common/fonts/`) จึงแสดงผลไทยถูกต้องบนคลาวด์

## ไฟล์สำคัญในชุดนี้
- `app.py` — โปรแกรมหลัก (เวอร์ชันคลาวด์ ไม่มี login)
- `requirements.txt` — แพ็กเกจสำหรับคลาวด์ (ไม่รวม pywebview ที่ใช้เฉพาะเดสก์ท็อป)
- `.gitignore` — กันไฟล์ที่ไม่ควรขึ้น GitHub
- `common/ app_pages/ modules/ data/ .streamlit/` — โค้ดและทรัพยากรของโปรแกรม (ชุดเดียวกับเวอร์ชันเดสก์ท็อป)
